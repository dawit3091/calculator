# calculator
Web Calculator
A sleek, fully interactive web calculator built from scratch using clean vanilla JavaScript logic and CSS Grid layout.

🚀 Features
Core Math Engine: Supports addition, subtraction, multiplication, and division.

State Toggle Switch: A fully functioning power button (OFF / ON) that locks/unlocks the entire application state.

Decimal Precision: Allows floating-point decimal math with built-in safety checks to prevent duplicate decimals.

Responsive Grid Layout: Built using modern CSS Grid and Flexbox for precise button alignment and mobile responsiveness.

🛠️ Tech Stack
HTML5: Structured semantics and custom data- attributes.

CSS3: CSS Grid Layout, Flexbox alignment, and interactive :hover/:active button states.

JavaScript (ES6+): Advanced DOM manipulation, event listeners (addEventListener), forEach loops, and dynamic state tracking.
